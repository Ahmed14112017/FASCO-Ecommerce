# FASCO E-commerce (Front-End)

A production-like E-commerce front-end application built with React / Next.js and TypeScript, integrated with a REST API.

---

## Features

- Authentication (Login / Register)
- Role-based access (User / Admin)
- Product listing & details
- Cart system (add / update / remove)
- Admin dashboard (CRUD products)
- Order flow integration
- API integration with real backend

---

## Tech Stack

- Next.js
- React
- TypeScript
- Redux Toolkit / Context API
- Tailwind CSS
- REST API

---

## Architecture

- Feature-based folder structure
- Reusable components
- Separated API layer
- Protected routes
- Role-based rendering
